01 — About

Who I Am

Catur Septian

@caturseptian · Yogyakarta, Indonesia

I'm a Full Stack Developer building production-grade web applications, REST APIs, and internal systems. I start with the fundamentals — HTML, CSS, JavaScript, PHP — then move into CodeIgniter & Laravel for robust backend frameworks. I also work with Node.js and Golang for systems that handle real load, stay maintainable, and ship reliably via Docker-based workflows.

My web development roots go back to freelance PHP work across SMBs, where I learned how applications actually break in production and how to keep them from doing so. That pragmatism shapes how I design backend services today: explicit contracts, minimal dependencies, and clear separation between concerns.

More recently I've expanded into data engineering — writing Python ETL pipelines, building Tableau integrations via the Hyper API, and automating reporting workflows that used to require manual effort. This gives me a useful full-picture perspective: I can build the system that generates the data and the pipeline that makes it useful.
